# Midjourney Background Prompts

Use these prompts for webtoon-style backgrounds inspired by Seongnam City Hall, Bundang-gu Office, Sujeong-gu Office, and Jungwon-gu Office.

Recommended Midjourney suffix for all prompts:

```text
--ar 16:9 --stylize 100 --v 6.1
```

Common direction:

```text
Korean webtoon style, realistic municipal office atmosphere, Seongnam city government mood, clean line art, subtle cel shading, calm cinematic lighting, no people, no readable text, no watermark
```

## Apply

After generating, export each image as JPG and place it into:

- `public/bg/<file-name>.jpg`

Examples:

- `public/bg/office_storage.jpg`
- `public/bg/civil_counter.jpg`
- `public/bg/executive_office.jpg`

The app is already wired to use these files automatically.
